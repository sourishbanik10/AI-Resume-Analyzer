# AI Resume Analyzer 🤖

An AI-powered resume analysis system that compares resumes with job descriptions and provides ATS scores, skill gap analysis, job matches, and AI suggestions.

## Features

- Resume parsing (PDF)
- Skill extraction
- ATS score calculation
- Role prediction
- Salary estimation
- AI resume improvement suggestions
- Job recommendation system
- Skill gap analysis

## Tech Stack

Python  
Streamlit  
NLP (Sentence Transformers)  
Scikit-learn  
FAISS  
OpenAI API

## Run the project

Install dependencies

pip install -r requirements.txt

Run the app

streamlit run app.py
